# 🚀 DevTrack

A Developer Project Management System built using **CodeIgniter 3, PHP, MySQL, and Bootstrap**.

DevTrack helps developers manage projects, tasks, deadlines, and progress status through a simple dashboard-based system.

## ✨ Features

- 🔐 User Login Authentication
- 📁 Project Management
- 📝 Task Management
- 🔄 Task Status Tracking
- 🔍 Search and Filter Tasks
- ⚠️ Project deletion protection when tasks are assigned
- 📊 Dashboard view

## 🛠️ Technologies Used

- PHP
- CodeIgniter 3
- MySQL
- Bootstrap 5
- HTML/CSS
- JavaScript

## 📂 Modules

### User Module
- Login system
- Session-based authentication

### Project Module
- Add projects
- Edit projects
- Delete projects
- View project details

### Task Module
- Create tasks
- Assign tasks to projects
- Update task status
- Manage deadlines
